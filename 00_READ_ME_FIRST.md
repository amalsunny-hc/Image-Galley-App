## 🎓 User Roles

### Regular User
```
Can:
  ✓ Register & login
  ✓ Upload images
  ✓ View public gallery
  ✓ Manage own images
  ✓ Set image visibility

Cannot:
  ✗ Manage other users
  ✗ Manage other images
  ✗ Access admin panel
```

### Admin User
```
Can:
  ✓ All user permissions PLUS:
  ✓ Manage all users
  ✓ Manage all images
  ✓ View dashboard
  ✓ Promote/demote users
  ✓ Delete users/images
  ✓ Toggle image visibility
```
